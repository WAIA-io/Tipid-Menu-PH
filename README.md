Tipid Menu PH 🇵🇭

Tipid Menu PH (formerly FoodHack) is a mobile-first, community-driven application designed to help Filipino foodies and budget-conscious consumers discover, share, and verify money-saving fast-food hacks and secret menu items.

🚀 Core Features

Interactive Map: Discover nearby restaurant spots with integrated navigation (Google Maps, Apple Maps, Waze).

Crowdsourced Hacks: View and share secret menu items and savings tips.

Official Promos: Stay updated with verified promotional deals from official brand channels.

Gamification: Earn points, level up through tiers (Newbie Saver to Tipid Legend), and build your profile.

PRO Tier: Unlock exclusive hacks and nationwide spots via loyalty points or a one-time upgrade.

Firebase Backend: Real-time data sync with Firestore and anonymous/secure authentication.

🛠️ Tech Stack

Framework: React.js

Styling: Tailwind CSS

Icons: Lucide React

Backend/DB: Firebase (Auth & Firestore)

Bundler: Vite
